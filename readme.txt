Allows you to easily add the WidgetBucks ad code to your Wordpress Sidebar\
\
== Description ==\
\
The WidgetBucks Sidebar Plugin is a simple plugin that is added as a widget. Once installed and activated you can add the UID of the WidgetBucks ad code and the widget will add the WidgetBucks widget to your sidebar.\
\
== Installation ==\
\
1. Install the Plugin\
2. Activate the Plugin\
3. Go to Appearance -> Widgets\
4. Add the widget to your sidebar\
5. Configure with Title and UID of WidgetBucks Ad Code\
6. Enjoy}
